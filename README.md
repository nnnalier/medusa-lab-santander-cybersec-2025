# Laboratório de Força Bruta com Medusa  
**Bootcamp Santander Cibersegurança 2025 — Entrega de Exercício**

## 1. Objetivo  
Descrever de forma documentada o experimento de força bruta usando Medusa em ambiente controlado.

## 2. Ambiente  
- Kali Linux bootável (atacante)  
- Metasploitable 2 (alvo vulnerável)  
- Rede interna isolada  

## 3. Metodologia  
1. Identificação do IP do alvo  
2. Descoberta de serviços ativos  
3. Criação de wordlists de usuários e senhas fracas  
4. Execução do Medusa para tentativa automática de autenticação  

## 4. Resultados & Análise  
- Credenciais fracas identificadas com sucesso  
- Demonstração de vulnerabilidade real em ambiente mal configurado  
- Evidência da eficácia de políticas de senha, MFA e segurança defensiva  

## 5. Lições e Boas Práticas  
- Uso de senhas fortes e únicas  
- Implementação de MFA  
- Restrição de acessos, monitoramento e hardening  
- Importância de auditoria contínua
